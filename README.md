## 📦 Installation
---
Install these and then copy my config files into your ~/.config folder

```bash
git clone https://github.com/tuusuario/dotfiles.git
cd dotfiles
./install.sh
```
## Optional dependencies
```
cbonsai neofetch pokemon-colorscripts-git feh jp2a 
```
## Terminal utils
```
nvim 
```
